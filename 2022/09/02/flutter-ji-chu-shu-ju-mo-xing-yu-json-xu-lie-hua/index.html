Flutter基础-数据模型与Json序列化

flutter json 序列化
Flutter
发布日期:   2022-09-02

一 flutter中如何进行json解析

1.1 dart中的数据模型

所有变量引用的都是 对象，每个对象都是一个 类 的实例。数字、函数以及 null 都是对象。除去 null 以外（如果你开启了 空安全）, 所有的类都继承于 Object 类

特殊类型 dynamic 将检查延迟到运行时进行,dynamic 这个类型不仅接受所有对象，也允许所有 operations。在编译时任何成员对 dynamic 类型值访问是允许的，但在运行时可能会引发异常,更多参考dart特殊类型：dynamic

  • Dart 语言支持下列内容：

  • Numbers (int, double)

  • Strings (String)

  • Booleans (bool)

  • Lists (也被称为 arrays)

  • Sets (Set)

  • Maps (Map)

  • Runes (常用于在 Characters API 中进行字符替换)

  • Symbols (Symbol)

  • The value null (Null)

1.2 dart中json解析

  • dart:convert 中使用内置的JSON解码器json.decode()来实现json解析，json.encode(object)来实现将对象转换为jsonString
  • json.decode()仅返回一个Map<String, dynamic> 或者List类型，这意味着直到运行时我们才知道值的类型。通过这种方法，我们失去了大部分静态类型语言特性：类型安全、自动补全和最重要的编译时异常。
    这样一来，我们的代码可能会变得非常容易出错。例如，当我们访问name或email字段时，我们输入的很快，导致字段名打错了。但由于这个 JSON 在 map 结构中，所以编译器不知道这个错误的字段名，所以编译时不会报错。
    因此实际开发中，我们会显示定义data model,并且声明对象字段为具体的类型，如String,int,需要将原始数据转换为Object对象
//一个JSON格式的用户列表字符串
String jsonStr='[{"name":"Jack"},{"name":"Rose"}]';`
//将JSON字符串转为Dart对象(此处是List)
List items=json.decode(jsonStr);
//输出第一个用户的姓名
print(items[0]["name"]);

1.3 如何手动实现数据模型与json之间的转换

  • 声明实体类User
class User {
   final String name;
   final String email;
   
   User(this.name, this.email);
   
   User.fromJson(Map<String, dynamic> json)
   : name = json['name'],
   email = json['email'];
   
   Map<String, dynamic> toJson() =>
     <String, dynamic>{
     'name': name,
     'email': email,
     };
}
  • json 序列化为User对象
Map userMap = json.decode(json);
var user = User.fromJson(userMap);
  • json反序列化
// json.encode 内部会自动调User.toJson()方法
String json = json.encode(user);

二 使用 json_serializable库实现json解析

上述例子中，我们通过手动声明data model,并实现字段映射，实现json解析，但是实际项目中，不可能为每个类都进行手动解析，这样成本太高，同时可能因手误导致的错误，在运行时才能发现。
因此我们将使用pub.dev 提供的json_serializable来辅助实现json解析

2.1 json_serializable集成与基础配置

  • pubspec.yaml 依赖配置
    dependencies:
  //1. json
  json_annotation: ^4.6.0

dev_dependencies:
  // 2. json
  build_runner: ^2.2.0
  json_serializable: ^6.3.1
     options:
        // 支持嵌套对象解析
        explicit_to_json: true

2.2 声明实体类及基础配置

  • 声明实体类Person 并注解@JsonSerializable,因为本例中Person的成员变量address也是对象，对嵌套对象解析，需要声明explicitToJson: true

    @JsonSerializable(explicitToJson: true)
class Person{

  Person({required this.name, required this.age, required this.address});
  // 设置序列化过程映射字段名
  @JsonKey(name: "real_name")
  String name;
  int age;
  
  Address address;
  
  factory Person.fromJson(Map<String,dynamic> json) => _$PersonFromJson(json);

  Map<String,dynamic> toJson() => _$PersonToJson(this);
}

  • 手动导入 part ‘example.g.dart’; 并添加fromJson(),toJson()方法

    part 'person.g.dart';
@JsonSerializable(explicitToJson: true)
class Person{
 
   Person({required this.name, required this.age, required this.address});
 
   String name;
   int age;
 
   Address address;
 
   // 1.
   factory Person.fromJson(Map<String,dynamic> json) => _$PersonFromJson(json);
   // 2.
   Map<String,dynamic> toJson() => _$PersonToJson(this);
}

3.3 FlutterJsonBeanFactory插件浅析

  • 经过前两步，已经生成实体类及具体json解析实现，如果此时修改了类，使用Alt+J 即可重新同步最新修改结果

  • generated/json/base 下包含了两个核心文件：json_field.dart，json_convert_content.dart

  • json_field.dart 解析：文件声明了两个类，JsonSerializable用于实体类注解，被标记的类会参与插件解析，JSONField用于实体类字段注解，用于设置json字段-实体类字段映射，
    及设置是否序列化与反序列，类似Java中transient关键字作用

class JsonSerializable{
  const JsonSerializable();
 }
 
 class JSONField {
  //Specify the parse field name
  final String? name;
  
  //Whether to participate in toJson
  final bool? serialize;
  
  //Whether to participate in fromMap
  final bool? deserialize;
  
  const JSONField({this.name, this.serialize, this.deserialize});
 }
  • json_convert_content.dart解析：
    1. 首先插件会根据标记过@JsonSerializable的注解注册对应的实体类，
    2. 然后json解析时会根据实体类的fromJson(Map<String,dynamic>)方法来实例化对象，并逐一调用jsonConvert.convert(json[‘key’])， 来实现每个实体类字段的解析和赋值，
    3. convert(dynamic) 会调用asT(dynamic)
    4. asT(dynamic) 会依次判断 T.toString的类型是”int”,”String”,”double”,”Map”… 然后对dynamic as T 转成对应的具体类型，如果是已经注册的具体实体类类型，则转换为对应的实体类类型（处理json对象嵌套对象的情况）
    5. 针对 List类型的json数据，通过fromJsonAsT(dynamic json)进行_getListChildType(json.map((e) => e as Map<String, dynamic>).toList()) ,做map转换一次对每个子内容进行fromJson()反序列化为具体的实体类
 import 'package:flutter/material.dart' show debugPrint;
 import 'package:github_client_app/models/address.dart';
 import 'package:github_client_app/models/person.dart';
 import 'package:github_client_app/models/weather.dart';
 
 JsonConvert jsonConvert = JsonConvert();
 typedef JsonConvertFunction<T> = T Function(Map<String, dynamic> json);

 class JsonConvert {
   static final Map<String, JsonConvertFunction> _convertFuncMap = {
     (Address).toString(): Address.fromJson,
     (Person).toString(): Person.fromJson,
     (Weather).toString(): Weather.fromJson,
   };

   T? convert<T>(dynamic value) {
    if (value == null) {
      return null;
    }
    return asT<T>(value);
   }

   List<T?>? convertList<T>(List<dynamic>? value) {
    if (value == null) {
      return null;
    }
    try {
      return value.map((dynamic e) => asT<T>(e)).toList();
    } catch (e, stackTrace) {
      debugPrint('asT<$T> $e $stackTrace');
      return <T>[];
    }
   }

   List<T>? convertListNotNull<T>(dynamic value) {
     if (value == null) {
      return null;
     }
     try {
       return (value as List<dynamic>).map((dynamic e) => asT<T>(e)!).toList();
     } catch (e, stackTrace) {
       debugPrint('asT<$T> $e $stackTrace');
       return <T>[];
     }
   }

   T? asT<T extends Object?>(dynamic value) {
      if (value is T) {
        return value;
      }
      final String type = T.toString();
      try {
        final String valueS = value.toString();
        if (type == "String") {
         return valueS as T;
        }else if (type == "int") {
         final int? intValue = int.tryParse(valueS);
         if (intValue == null) {
           return double.tryParse(valueS)?.toInt() as T?;
          }else {
           return intValue as T;
          }
        }else if (type == "double") {
          return double.parse(valueS) as T;
        }else if (type == "DateTime") {
          return DateTime.parse(valueS) as T;
        }else if (type == "bool") {
          if (valueS == '0' || valueS == '1') {
            return (valueS == '1') as T;
          }
          return (valueS == 'true') as T;
        }else if (type == "Map" || type.startsWith("Map<")) {
         return value as T;
        }else {
          if (_convertFuncMap.containsKey(type)) {
           return _convertFuncMap[type]!(value) as T;
          }else {
           throw UnimplementedError('$type unimplemented');
          }
        }
      } catch (e, stackTrace) {
        debugPrint('asT<$T> $e $stackTrace');
        return null;
      }
   }

//list is returned by type
static M? _getListChildType<M>(List<Map<String, dynamic>> data) {
	if(<Address>[] is M){
		return data.map<Address>((Map<String, dynamic> e) => Address.fromJson(e)).toList() as M;
	}
	if(<Person>[] is M){
		return data.map<Person>((Map<String, dynamic> e) => Person.fromJson(e)).toList() as M;
	}
	if(<Weather>[] is M){
		return data.map<Weather>((Map<String, dynamic> e) => Weather.fromJson(e)).toList() as M;
	}

	debugPrint("${M.toString()} not found");

	return null;
   }

static M? fromJsonAsT<M>(dynamic json) {
	if (json is List) {
		return _getListChildType<M>(json.map((e) => e as Map<String, dynamic>).toList());
	} else {
		return jsonConvert.asT<M>(json);
	}
}
}
